Simulating a Forest Fire with python - 2020

The GIF above shows my simulation of a forest fire which was the subject of my final year physics coding project. The green squares represent trees, the black squares empty sites, and the orange squares burning trees. The Drossel and Schwabel model was used in this simulation which is an application of site percolation theory. Site percolation theory has numerous important applications from computer networks to electrical conductivity.

Cluster labelling was implemented into the project as well (a cluster is a group of trees connected by their sides) via the Hosehen-Kopelman algorithm. This algorithm is an implementation of the union-find algorithm, which is important in Computer Science.

This project taught me a lot about algorithms and the importance of simulations and data analysis to solve problems which cannot be solved analytically.

Calculating the speed of light with Python - 2020

For my physics degree year 2 lab project, we were tasked with calculating the speed of light in air. The basic idea was to move a transmitted laser various distances from a reference laser and measuring the resulting time difference between their peaks.

Initially we only had values of the x and y positions of the transmitted and reference laser. We developed a python program to use data analysis (least squares fit, root finding) to calculate the speed of light in air from this data.
